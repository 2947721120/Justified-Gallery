Infinite Scroll

In this example you can see how to use the plugin to obtain an infinite scroll.

The first thing to do is to initialize the gallery as usually.

{% highlight javascript %} $('#gallery').justifiedGallery({rowHeight:120}); {% endhighlight %}

Then you need to implement the infinite scroll as you want. After that you added some images you just have to recall Justified Gallery again. For example:

{% highlight javascript %} $(window).scroll(function() { if($(window).scrollTop() + $(window).height() == $(document).height()) { for (var i = 0; i < 5; i++) { $('#gallery').append('' + '' + ''); } $('#gallery').justifiedGallery('norewind'); } }); {% endhighlight %}

You can notice that we are calling Justified Gallery using the special parameter 'norewind'. In this way the plugin doesn't re-analyzes all the images, but only the new ones. This parameter increases the performance, and also prevents the page to have strange behaviours with the scrollbars.
