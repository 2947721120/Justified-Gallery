Justified Gallery documentation

Basic example

Here we have a div, with the id myExample0, that has a list of links that point to some images (original size images), and where inside of each link there is a thumbnail. Calling Justified Gallery, the disposition of all the thumbnails is changed to fill all the spaces (justification). The images can be changed: bigger or smaller thumbnails can be loaded to always have an high quality.

The plugin knows the path of the other thumbnails of the same image changing the filenames suffix. For example, if in a link there is the thumbnail path/to/image1_t.jpg, the plugin can be load the thumbnails path/to/image1_b.jpg. The default suffixes are the same of Flickr (and this permits a smooth integration with this service).

Remember that the default value for the rowHeight is 120. For this example the thumbnails with a longest side of 240px (i.e. _m suffix) is a good choice to have be fast (i.e. the plugin doesn't need to reload the majority of all thumbnails). 

<div id="basicExample">
<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
	<img alt="What's your destination?" src="photos/8083451788_552becfbc7_m.jpg" />
</a>
<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
	<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_m.jpg" />
</a>
...
</div>

$("#basicExample").justifiedGallery();

Row heights, not justify last row, margins

This is an example where we use smaller images (rowHeight : 70). To be faster, in this case we use smaller thumbnails (i.e. the _t suffix, longest side of images is less than 100px). In this way the plugin probably will not load other thumbnails.

Remember that the rowHeight is an approximation for the row heights. The row heights can be higher to reduce the image crops. You can also set a maximum value for the height, using the option maxRowHeight.

Furthermore, note that in this example we don't justify the last row (justifyLastRow : false), and also that we are using a margin of 5px (margins : 5px). 

<div id="myExample1">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$('#myExample1').justifiedGallery({
	rowHeight : 70,
	justifyLastRow : false, 
	margins : 3
});

Lightbox

In this example we use the Colorbox lightbox. To do this we create a listener for the callback jg.complete callback function where we initialize Colorbox. Note that we are also using the rel = 'gallery2' option, that sets to 'gallery2' the rel attribute for all the links. In this way Colorbox can create a group of images. 

<div id="lightboxExample">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$('#lightboxExample').justifiedGallery({
	justifyLastRow : false, 
	rowHeight : 70, 
	rel : 'gallery2', //replace with 'gallery2' the rel attribute of each link
	margins : 1
}).on('jg.complete', function () { //callback function when the gallery has been created
	$(this).find('a').colorbox({
		maxWidth : '80%',
		maxHeight : '80%',
		opacity : 0.8,
		transition : 'elastic',
		current : ''
	});
});

Captions

The captions are automatically created using the alt attribute of images and, if it doesn't exist, the title attribute of links. To have custom captions you can create a div attribute after the img tag. This one must have the class caption to be valid. 

<div id="captionsExample">
	<a href="photos/8083451788_552becfbc7_b.jpg">
		<img src="photos/8083451788_552becfbc7_m.jpg" />
		<div class="caption">
			I'm <span style="color: red">a</span> <strong>custom</strong> caption!<div>I'm a div</div>
		</div>
	</a>
	<a href="photos/8083451788_552becfbc7_b.jpg" title="I'm in the title">
		<img alt="I'm in the alt" src="photos/8083451788_552becfbc7_m.jpg" />
	</a>
	...
</div>

$('#captionsExample').justifiedGallery();

Fixed Height, no caption, target _blank

In this example you can see how the fixed height option works (fixedHeight : true). Note that the big images is cutted, because they have a fixed height that they need to respect. Also, we disable the captions (captions : false), and then we set the images to be opened in a new page (i.e. each link target is set to '_blank' using the option target : '_blank'). 

<div id="myExample2">
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_m.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_m.jpg" />
	</a>
	...
</div>

$('#myExample2').justifiedGallery({
	rowHeight : 200, 
	fixedHeight : true,
	captions : false,
	target : '_blank',
	margins : 2
});

Not Found images

Test with a gallery of 5 images, where one of them is inexistent. This image is ignored and is not shown in the gallery. Furthermore here we have a thumbnail, but in the server we don't have the other thumbnails. In this case the plugin doesn't load the other thubmnails and the image is shown with its low quality. This permits a very clean and silent way to manage errors without hurting the user (the errors are shown in the console).

One Size Images

You need to have six thumbnails for all image? It' not mandatory! For example you may have only one type of thumbnail (e.g. "myimage_t.jpg") and its original image (e.g. "myimage_b.jpg").

This example show you how to use the Justified Gallery with only one thumbnail (in this case only the one with _t as suffix). This can be done modifying the sizeRangeSuffixes option to be the same for all the sizes. 

<div id="oneSizeExample" >
	<a href="photos/8083451788_552becfbc7_b.jpg" title="What's your destination?">
		<img alt="What's your destination?" src="photos/8083451788_552becfbc7_t.jpg" />
	</a>
	<a href="photos/7948632554_01f6ae6b6f_b.jpg" title="Just in a dream Place">
		<img alt="Just in a dream Place" src="photos/7948632554_01f6ae6b6f_t.jpg" />
	</a>
	...
</div>

$('#oneSizeExample').justifiedGallery({
	sizeRangeSuffixes : {
		lt100 : '_t', 
		lt240 : '_t', 
		lt320 : '_t', 
		lt500 : '_t', 
		lt640 : '_t', 
		lt1024 : '_t'
	},
	rowHeight : 250
});
